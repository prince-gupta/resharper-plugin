Resharper-Plugin

A Custom Plugin to parse resharper xml generated as the result of static code analysis.

About.

This project is to develop Sonar Qube plugin for re-sharper. Which can parse the resultant xml after static code analysis by re-sharper.

How compile it

Must have

  • Java 7 or above
  • Maven preferably 3.2.3 or above

Steps

  1. Clone or download repository to desired folder.
  2. Navigate to repository folder.
  3. Open command prompt.
  4. Run mvn clean compile package
  5. Jar will get created in target folder.

How to use it

Must have

  • Sonar Scanner 2.8

Steps

  1. Copy generated XML from Resharper to a folder of your choice.
  2. Rename it to start with "rs".

  3. Create a file named as sonar-project.properties.

    sonar-project.properties

    sonar.projectKey=<key-for-project for sonar-db>

    sonar.projectName=<name-for-project that will display on dashboard>

    sonar.projectVersion=<project-version>

    sonar.sources=<source-directory-path-relative-to-property-file>

    sonar.resharperplugin.reportsPath=<directory-path-having-resharper-xml-relative-to-property-file>/

  4. Run sonar-scanner command, it will sends the xml to plugin on server and server will parse it and will show results.

    • If you want to send xml report to remote server or any other machine hosting Sonar Qube , do following steps :
      • Update the global settings (server URL) by editing <install_directory>/conf/sonar-scanner.properties: Default SonarQube server sonar.host.url=http://localhost:9000
  5. On Sonar Qube add a widget with this plugin named as Resharper Analysis Widget.
  6. After adding widget and refreshing the page plugin will get displayed to dashboard home.
